# Apex API Query
Apex Legends API 查询插件

## 使用方法
在 `nonebot-plugin-apex-api-query.py` 中的第 10 行填入您的 API 密钥。
您可以点击 [此处](https://portal.apexlegendsapi.com/) 申请 API 密钥。
在 Bot 私聊 或 已加入的群聊 中发送命令。

### 查询玩家信息
`/bridge [玩家名称]` 、 `/玩家 [玩家名称]`
暂不支持 UID 查询。

输出示例：

### 查询大逃杀地图轮换
`/maprotation` 、 `/地图`

```text

下个地图: 世界尽头 
剩余时间: 00:06:27 
```
